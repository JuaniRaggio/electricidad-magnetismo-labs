# Electricidad y Magnetismo - Laboratorios

## Curvas Caracteristicas I-V

Dashboard interactivo: https://curvas-caracteristicas.streamlit.app/

Analisis de curvas I-V de resistencia, diodo, lampara incandescente y LED. Incluye ajuste de modelos, simulacion Monte Carlo, resistencia dinamica, temperatura del filamento, factor de idealidad y energia del gap.
